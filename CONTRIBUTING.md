# Contributing to ClawSnap

Thanks for your interest in contributing!

## Workflow

1. Fork this repository
2. Create a feature branch
3. Make focused changes
4. Open a Pull Request with:
   - What changed
   - Why it changed
   - Screenshots (if UI changed)

## Guidelines

- Keep changes small and reviewable
- Prefer readable vanilla JS
- Avoid introducing build tooling unless clearly needed
- Do not commit secrets (bot tokens, chat IDs)

## Local testing checklist

- [ ] Extension loads in `chrome://extensions`
- [ ] Capture works on common websites
- [ ] Telegram sendPhoto succeeds
- [ ] Telegram sendMessage succeeds
- [ ] Template variables render as expected
