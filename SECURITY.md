# Security Policy

## Supported versions

Latest `main` is supported.

## Reporting a vulnerability

Please avoid public disclosure before a fix is available.

When reporting, include:
- Steps to reproduce
- Impact assessment
- Suggested mitigation (if available)

## Security notes

- Bot token is sensitive and should be rotated if leaked.
- Do not paste production bot tokens in issue threads.
- Consider using a dedicated Telegram bot for this extension.
